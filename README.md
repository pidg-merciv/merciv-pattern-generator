# Brand Pattern Generator

A customizable dot pattern generator for creating unique brand patterns. Built with vanilla JavaScript and HTML5 Canvas.

## Features

- **Adjustable Dot Size**: Control the size of individual dots (2-30px)
- **Density Control**: Adjust pattern density (0-100%)
- **Seeded Randomization**: Generate reproducible patterns with custom seeds
- **Color Customization**: Choose foreground and background colors
- **Flexible Canvas Size**: Generate patterns from 200px to 2400px
- **Export Options**: Download as PNG or SVG

## Usage

Simply open `pattern-generator.html` in your web browser. No build process or dependencies required.

## Controls

- **Dot Size**: Adjusts the diameter of each dot
- **Density**: Controls how many dots appear in the pattern
- **Random Seed**: Sets the seed for reproducible random patterns
- **Colors**: Customize foreground (dots) and background colors
- **Canvas Size**: Set the output dimensions
- **Regenerate**: Refresh the pattern with current settings
- **Download PNG/SVG**: Export your pattern in either format
